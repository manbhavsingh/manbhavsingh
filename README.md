## 🏆 Featured Projects

### 🧊 3D Face Reconstruction under Occlusion
> *Differentiable rendering · BiSeNet · 3DMM · MediaPipe · GAN · PyTorch3D*

Designed an end-to-end pipeline for **robust 3D face reconstruction from heavily occluded 2D images**. Integrated BiSeNet-based occlusion segmentation with a 3D Morphable Model (3DMM) fitted via differentiable rendering, guided by a custom **occlusion-aware loss function**. GAN-based refinement ensures photorealistic surface recovery even under severe partial occlusion.

---

### 🏥 Hybrid ML Diagnostic Pipeline — DDXPlus
> *XGBoost · Ontology-aligned Fuzzy Matching · Clinical NLP · 1.3M patient records*

Engineered a **clinical diagnostic system achieving 99.72% accuracy across 49 diseases** on the full DDXPlus dataset (1.3M patient records). The pipeline combines XGBoost classification with ontology-aligned fuzzy symptom matching to handle noisy, real-world medical inputs.

**📈 Outperformed all published RL-based baselines:**

| CASANDE (RL) | Baseline |

| **Ours (Hybrid ML)** | **99.72%** |

### 🧬 Explainable Alzheimer's Detection from MRI
> *ResNet · Grad-CAM · PyTorch · Medical Imaging*

Built an **interpretable deep learning classifier** for Alzheimer's stage detection from structural MRI scans. Grad-CAM visualizations highlight clinically meaningful brain regions (hippocampus, entorhinal cortex), making model decisions transparent and auditable by medical professionals.

---

### 💪 Smart Gym Rep Counter
> *ESP32 · Arduino · Smartphone Accelerometer · Real-time Edge ML*

Deployed a **real-time exercise classification and rep-counting system** on ESP32 microcontroller using smartphone accelerometer data over BLE. Classifies multiple exercise types in real time with low-latency inference — no cloud dependency, fully edge-native.
